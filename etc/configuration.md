## Configuration

bl uses `~/.bl.yml` for configuration.

```
bl init
$EDITOR ~/.bl.yml
```

### .bl.yml Parameters

```
:space_id: '***'    # your backlog space id
:api_key: '***'     # your backlog api key
:project_key: '***' # your backlog project key
:issue:
  :default:         # issue default parameters(add command use this for easiness)
    :projectId:
    :issueTypeId:
    :priorityId:
    :assigneeId:
```
